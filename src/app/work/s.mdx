## Overview

Data Pipeline Automation is a configurable backend framework designed to automate
schema creation, synthetic data generation, and multi-format data delivery.

## Key Features

- Schema-driven database generation from JSON models
- Generated 150,000+ realistic synthetic records using Faker
- Maintained referential integrity across PostgreSQL, CSV, JSON, and APIs
- Deployed FastAPI server with pagination, filtering, and analytics
- Integrated with Supabase PostgreSQL for cloud storage and migrations

## Tech Stack

Python · FastAPI · PostgreSQL (Supabase) · Pandas · Faker
