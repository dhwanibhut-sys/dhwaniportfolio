## Overview

Skill Spell is an AI-driven learning platform designed to provide **personalized learning
roadmaps, adaptive assessments, and intelligent educational assistance** tailored to
individual learners.

The platform focuses on reducing friction in self-learning by dynamically curating
resources, answering contextual queries, and adapting content based on user progress.

## Key Features

- **Personalized Learning Roadmaps**  
  Designed a custom roadmap generator that creates structured learning paths with curated
  resources such as articles, books, and lectures.

- **Context-Aware AI Chatbot (RAG)**  
  Implemented a Retrieval Augmented Generation system enabling users to:
  - Ask concept-level questions  
  - Generate summaries of YouTube videos (via Whisper.cpp)  
  - Query uploaded documents using OCR and vector search (ChromaDB)

- **Real-Time Knowledge Retrieval**  
  Integrated dynamic web scraping using Crawl4AI and a self-hosted XNG browser to fetch
  up-to-date learning content.

- **Secure, Modular Deployment**  
  Deployed microservices in isolated Docker containers to ensure scalability, security,
  and clean separation of concerns.

## Technologies Used

- **AI / ML**: TensorFlow, LangChain, OpenAI, Ollama  
- **Data & Retrieval**: ChromaDB, OCR pipelines  
- **Frontend**: Streamlit  
- **Infrastructure**: Docker, self-hosted browser services  
- **Speech & Media**: Whisper.cpp

## Context

Skill Spell was built during **HackSpire**, a 24-hour hackathon organized by ASPDC at  
Adani University, Ahmedabad.

## Outcome

The project demonstrated how AI-driven personalization and retrieval systems can
significantly improve the learning experience. Skill Spell highlights strong capabilities
in **LLM integration, prompt engineering, system design, and rapid prototyping under
time constraints**.
